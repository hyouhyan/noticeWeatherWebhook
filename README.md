# noticeWeatherWebhook

1時間おきに広島の天気をWebhookでDiscordに通知するGAS

---

今度、2泊3日で広島旅行に行くんすよ。  
ただ一つ問題があって、天気によって予定が変わる可能性があるんです。  
飯屋の予約とかもあるし、こまめに天気予報を確認したい。
でも、ウェザーニュースを毎日開くのがめんどくさい。

ということで、こんな感じのbotを作ってみました。
